# App Demo Flow

Use this as the README demo script, QA checklist, or product walkthrough.

## Admin

1. Sign in with `admin@smarthr.com` and demo password.
2. Open Dashboard to review attendance, KPI, alerts, and task summaries.
3. Open Staff and create/edit an employee.
4. Set `Employee Range (meters)` and `Daily Break Limit (minutes)`.
5. Assign a task card to one employee or all employees.
6. Approve/reject overtime, leaves, expenses, and attendance edit requests.
7. Export attendance or staff reports where available.

## Staff

1. Sign in with `staff@smarthr.com`.
2. Open Attendance.
3. Inside assigned range and matching office Wi-Fi, perform Check In/Out with selfie.
4. Outside assigned range, confirm Check In/Out is blocked and only Mark Visit / Start Break is shown.
5. Open Tasks and complete assigned task cards.
6. Review KPI, salary, loan, leave, expense, notifications, and profile details.

## Supervisor

1. Sign in with `supervisor@smarthr.com`.
2. Review team attendance and staff list scoped to supervisor branch/team.
3. Approve or reject leave/expense/overtime requests for the scoped team.
